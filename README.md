## Description

Calorie tracker allows users to create foods/recipes, calculate nutrition information by inputting weight values, and track important statistics such as calories and protein throughout the day.

## Tech Stack

- React.js
- Node.js
- Bootstrap
- Firebase

## Version History

### v1
